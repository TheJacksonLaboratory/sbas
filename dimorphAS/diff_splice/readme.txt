

significant_events.zip – the IDs of significant events in each tissue for each event type.  These IDs correspond to the ‘fromGTF‘ files.

number_of_tissues_per_gene.txt – the number of tissues in which each gene is significant

number_of_genes_per_tissue.txt – the number of genes (not events) that are significant for each tissue

GO_enrichment.txt – enriched GO categories that were detected by Ontologizer
